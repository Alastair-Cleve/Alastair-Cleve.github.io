Mapping in JavaScript & C#

JavaScript provides Array.prototype.map(), i.e. a method on the Array class that returns a new modified array: 


      var array = [1, 2, 3];
      array = array.map(function(element) {
        return element * 2;
      });

      console.log(array);
      // [2, 4, 6]

Achieving the same functionality in C# depends on the data structure. If you are using System.Array, you can use a LINQ extension method with lambda syntax, whereas if you are using System.Collections.Generic.List<T>, you can call ConvertAll<T>.

Array 


      using System;
      using System.Linq;

      int[] array = new int[] { 1, 2, 3 };
      array = array.Select(element => element * 2).ToArray();

      foreach (int el in array) {
        Console.WriteLine(el);
      }
      // 2, 4, 6

List<T> 


      using System;
      using System.Collections.Generic;

      List<int> array = new List<int> { 1, 2, 3 };
      array = array.ConvertAll<int>(element => element * 2);

      foreach (int el in array) {
        Console.WriteLine(el);
      }
      // 2, 4, 6
